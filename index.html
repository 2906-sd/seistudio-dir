HOW_IT_WORKS.MD

How does this site function?

The main version of this site runs on Vercel and features a built-in administrative panel. This panel allows me to instantly publish and manage content without manually editing any HTML or CSS files. (I haaaaate scrolling through lines of code, lol.)

All underlying text and content data (all my art, logs, letters, shrines, and even the bulletin board messages) are kept secure via Firebase, while the codebase is managed through GitHub. Whenever I push an update or add a post on the Vercel side, an automated GitHub YML workflow triggers to compile those changes and push a static mirror over to Neocities!.

Note on Neocities Login: If you are viewing this on Neocities and tried logging into the admin portal, it will not work. :p Because Neocities only hosts a static copy of the site, the secure serverless authentication routines only execute on the primary Vercel deployment.

Resources:

  • Icons: Streamline Pixel
  • Hosting & Optimization: Cloudinary (used for icon and image hosting)
  • Fonts: Comfortaa, Newsreader, and Space Mono
